Como Usar

Este visualizador permite que você explore modelos 3D diretamente no seu navegador. Siga os passos:

  1. Clique no botão **"Escolher Arquivo 3D"** acima.
  2. Selecione um arquivo de modelo 3D no formato **.glb** ou **.gltf** do seu computador.
  3. O modelo será carregado automaticamente na área 3D.
  4. Use o **mouse** para interagir:
    • **Clique e arraste:** Para girar o modelo.
    • **Scroll do mouse (rodinha):** Para dar zoom no modelo.
  5. Você também pode baixar os modelos de exemplo abaixo para testar!
  6. Use os botões abaixo da área 3D para **alternar os eixos de referência**, **mostrar/esconder a base/chão** ou **mudar a cor de fundo** da cena.
